# Bjarne Stroustrup – C++ Book Practice

This repository is dedicated to Bjarne Stroustrup’s “Programming: Principles and Practice Using C++, Second Edition”.

It contains my programming exercises and practice solutions, organised by chapter.

# Repository Structure
Each chapter from the book has its own folder
Each folder contains solutions to drills, exercises, and questions from that chapter
All code is written in C++

# Purpose
Reinforce concepts from the book
Practice problem-solving in C++
Keep a structured record of chapter-based exercises

#How to Use
Clone or download the repository
Navigate to a chapter folder
Open the C++ files in your preferred IDE or compiler
Compile and run the code to test the solutions

# Notes
Files are named according to the exercises they correspond to
Make sure your environment supports C++11 or later, as some examples use modern C++ features
